**Pre-Installation**
- install rust if you dont have
- add this line to your shellrc file -> `export PATH="$HOME/.cargo/bin:$PATH"`

**Installation**
- `git clone https://github.com/salandarman/jewfetch.git`
- `cd jewfetch`
- `chmod +x install.sh`
- `./install.sh`
